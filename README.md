# Bharatanatyam Mudra Recognition App

This project uses YOLOv8 classification to recognize Bharatanatyam hand gestures (mudras) in images and live webcam feed.

## Features
- Train YOLOv8 classification model on your mudra dataset
- Predict mudras on single images
- Live mudra detection using webcam with smoothing
