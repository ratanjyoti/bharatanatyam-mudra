# Bharatanatyam Mudra Recognition App

This project uses YOLOv8 classification to recognize Bharatanatyam hand gestures (mudras) in images and live webcam feed.

## Features
- Train YOLOv8 classification model on your mudra dataset
- Predict mudras on single images
- Live mudra detection using webcam with smoothing

## Installation
```bash
git clone https://github.com/your-username/bharatanatyam-mudra-app.git
cd bharatanatyam-mudra-app
pip install -r requirements.txt



## Running Mudra Detection

To run mudra detection on images, execute the Jupyter Notebook:
jupyter notebook run_for_detecting.ipynb


This notebook loads the trained YOLOv8 mudra classification model and runs detection on sample images or your own images.

Alternatively, you can run the detection script directly if available.

---

This setup helps you quickly get started with training and inference on the Bharatanatyam Mudra dataset using YOLOv8 classification.
